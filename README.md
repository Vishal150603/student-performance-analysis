# Student Performance Analysis

## Project Overview
This project analyzes student performance data using Python and Jupyter Notebook. The objective is to identify factors affecting students' academic performance and gain meaningful insights through data visualization and statistical analysis.

## Dataset
The dataset contains information about students, including:

- Gender
- Ethnic Group
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

Dataset File:
- Expanded_data_with_more_features.csv

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Data Analysis Performed

### Data Cleaning
- Checked missing values
- Removed duplicate records
- Verified data types

### Exploratory Data Analysis (EDA)
- Gender distribution analysis
- Ethnic group analysis
- Parental education impact
- Lunch type impact on scores
- Test preparation course effectiveness
- Score distribution analysis

### Visualizations
- Count Plots
- Bar Charts
- Histograms
- Box Plots
- Heatmaps

## Key Insights

- Students who completed test preparation courses generally scored higher.
- Parental education level showed a positive impact on student performance.
- Significant differences were observed among ethnic groups.
- Reading and writing scores were strongly correlated.
